Java基础1--static

Posted on 2015-05-16   |   In computer technology   |  

什么是static

        ”static” 是java中的关键字，中文的意思是静态的。它可以被用来修饰成员变量，成员方法或者代码块。被static修饰的成员变量和成员方法独立于该类的任何对象。也就是说，它不依赖类特定的实例，被类的所有实例共享。只要这个类被加载，Java虚拟机就能根据类名在运行时数据区的方法区内定找到他们。因此，static对象可以在它的任何对象创建之前访问，无需引用任何对象。

如何使用static

1.static 修饰的成员变量或成员方法
        当一个类成员变量或者类方法被声明为public static 时，这个成员变量或者方法不需要被实例化就可以被其他类通过类名调用。但是如果是private static，我们依旧需要在另一个类中实例化一个对象之后再使用这个变量或方法。例如：

public class countStudent{
    public static int num;
    public String name;
    public countStudent(String name){
      this.name=name;
    }
    public static int getNum(){
    return num;
    }
    public String getName(){
    return this.name;
    }
    ……………..
}
public class test{
    public static void main(String[] args){
        System.out.println(countStudent.num);
        countStudent a=new countStudent(name);
        System.out.println(a.name);
    }
}

2.static类方法需要注意什么
        被声明为static的类方法可以直接调用本类中的被声明为static的类成员变量或者成员方法，但是不能直接调用非static变量和方法，如果要调用，必须先实例化一个对象，然后再调用。非static类方法可以直接调用static和非static类成员。

3.static代码块
        static 代码块也叫静态代码块，是在类中独立于类成员的static语句块，可以有多个，位置可以随便放，它不在任何的方法体内，JVM加载类时会执行这些静态的代码块，如果static代码块有多个，JVM将按照它们在类中出现的先后顺序依次执行它们，每个代码块只会被执行一次。static代码块会优于非static代码块被先执行。同时静态初始化块只能给静态变量赋值，不能初始化普通的成员变量。

public class HelloWorld {
        String name; // 声明变量name
        String sex; // 声明变量sex
        static int age;// 声明静态变量age
        // 构造方法
        public  HelloWorld () {
        System.out.println(“通过构造方法初始化name”);
        name = “tom”;
    }
        // 初始化块
        {
            System.out.println(“通过初始化块初始化sex”);
            sex = “男”;
        }
        // 静态初始化块
         static{
        System.out.println(“通过静态初始化块初始化age”);
        age = 20;
        }
        public void show() {
            System.out.println(“姓名：” + name + “，性别：” + sex + “，年龄：” + age);
        }
        public static void main(String[] args) {
            // 创建对象
            HelloWorld hello = new HelloWorld();
            // 调用对象的show方法
            hello.show();
            HelloWorld hh=new HelloWorld();
            hh.show();
        }
 }

4.static final
        static final用来修饰成员变量和成员方法，可简单理解为“全局常量”！对于变量，表示一旦给值就不可修改，并且通过类名可以访问。对于方法，表示不可覆盖，并且可以通过类名直接访问。
